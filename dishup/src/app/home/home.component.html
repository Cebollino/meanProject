Hello, world!

La idea de esta pagina web es manejar la informacion interna de un bar/restaurante.

Esta aplicacion esta creada con Angular y bootstrap para la parte del cliente, mientras los encargados de trabajar en la parte del servidor seran:

  • NodeJs usando el framework express para un manejo de node mas comodo, este sera el intermediario entre las peticiones del cliente y la base de datos.
  • MongoDB que manejare a traves de mongoose para hacerlo mas ameno.
